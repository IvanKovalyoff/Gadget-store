# Gadget Store

**Live demo:** https://ivankovalyoff.github.io/Gadget-store/  
**Repository:** https://github.com/IvanKovalyoff/Gadget-store

Gadget Store is a single-page application (SPA) that represents a product catalog with dedicated product pages.
The project was created as a portfolio project with a strong focus on architecture, scalability, and code quality.

---

## Features

- Product catalog with reusable UI components
- Product details pages
- Client-side routing
- Internationalization (i18n)
- Responsive layout
- Storybook for UI components
- CI/CD deployment via GitHub Actions
- Code quality tools (ESLint, Prettier)

---

## Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **React Router**
- **i18next**
- **SCSS**
- **Storybook**
- **GitHub Actions**

---

## Project Structure

src/
├─ components/ # Reusable UI components
├─ pages/ # Page-level components
├─ context/ # React Context providers
├─ hooks/ # Custom React hooks
├─ utils/ # Helper functions
├─ types/ # TypeScript types
├─ styles/ # Global styles
├─ stories/ # Storybook stories

## Getting Started

### Requirements
- Node.js 20+
- npm

### Install & run locally
```bash
npm install
npm run dev
```
## Build
```bash
npm run build
npm run preview
```
## Code Quality
```bash
npm run lint
npm run format
```
## Storybook
```bash
npm run storybook
```


## CI / Deployment
The project is automatically built and deployed to GitHub Pages using GitHub Actions on every push to the main branch.
